字符流中第一个不重复的字符

问题描述

请实现一个函数用来找出字符流中第一个只出现一次的字符。例如，当从字符流中只读出前两个字符”go”时，第一个只出现一次的字符是”g”。当从该字符流中读出前六个字符“google”时，第一个只出现一次的字符是”l”。
如果当前字符流没有存在出现一次的字符，返回#字符。

问题分析

方法1：蛮力轮询，从头开始扫描，拿每个字符和后面字符依次比较，时间复杂度O(n2)。
方法2：使用哈希表来统计每个字符在该字符串出现的次数，key是字符，value是该字符出现的次数。时间复杂度O(n)。

两个链表的第一个公共结点

问题描述

输入两个链表，找出它们的第一个公共结点。

问题分析

1、蛮力查找，如果第一个链表长度为m，第二个链表长度为n，则该算法时间复杂度是o(mn)。不推荐。
2、借用两个辅助栈，将两个链表分别压入栈中，这样两个链表的尾节点就位于两个栈的栈顶，接下来比较两个栈顶结点，如果相同则弹出继续比较，直到找到最后一组相同的节点。
时间复杂度O(m+n)，空间复杂度O(m+n)。
3、先遍历两个链表得到他们的长度，然后在长链表上先进行遍历，直到长度和短链表相同，再将两个链表一起开始遍历，找到第一个共同结点。
时间复杂度O(m+n)，但不需要辅助栈。

不用加减乘除做加法

发表于 2016-03-17 | 分类于 剑指Offer

不用加减乘除做加法

问题描述

写一个函数，求两个整数之和，要求在函数体内不得使用+、-、*、/四则运算符号。

问题分析

对数字做运算，除了四则运算之外，只剩下位运算了。位运算是针对二进制的，考虑把二进制的加法用位运算来代替。
第一步：不考虑进位对每一位相加，等同于异或^操作。
第二步：考虑进位，使用与&操作，然后左移一位进位。
第三步：把前两步的结果相加，直到不产生进位为止
