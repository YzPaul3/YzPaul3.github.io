字符流中第一个不重复的字符

问题描述

请实现一个函数用来找出字符流中第一个只出现一次的字符。例如，当从字符流中只读出前两个字符”go”时，第一个只出现一次的字符是”g”。当从该字符流中读出前六个字符“google”时，第一个只出现一次的字符是”l”。
如果当前字符流没有存在出现一次的字符，返回#字符。

问题分析

方法1：蛮力轮询，从头开始扫描，拿每个字符和后面字符依次比较，时间复杂度O(n2)。
方法2：使用哈希表来统计每个字符在该字符串出现的次数，key是字符，value是该字符出现的次数。时间复杂度O(n)。

发表于 2016-03-18 | 分类于 剑指Offer

两个链表的第一个公共结点

问题描述

输入两个链表，找出它们的第一个公共结点。

问题分析

1、蛮力查找，如果第一个链表长度为m，第二个链表长度为n，则该算法时间复杂度是o(mn)。不推荐。
2、借用两个辅助栈，将两个链表分别压入栈中，这样两个链表的尾节点就位于两个栈的栈顶，接下来比较两个栈顶结点，如果相同则弹出继续比较，直到找到最后一组相同的节点。
时间复杂度O(m+n)，空间复杂度O(m+n)。
3、先遍历两个链表得到他们的长度，然后在长链表上先进行遍历，直到长度和短链表相同，再将两个链表一起开始遍历，找到第一个共同结点。
时间复杂度O(m+n)，但不需要辅助栈。

不用加减乘除做加法

问题描述

写一个函数，求两个整数之和，要求在函数体内不得使用+、-、*、/四则运算符号。

问题分析

对数字做运算，除了四则运算之外，只剩下位运算了。位运算是针对二进制的，考虑把二进制的加法用位运算来代替。
第一步：不考虑进位对每一位相加，等同于异或^操作。
第二步：考虑进位，使用与&操作，然后左移一位进位。
第三步：把前两步的结果相加，直到不产生进位为止

二叉树的深度

问题描述

输入一棵二叉树，求该树的深度。从根结点到叶结点依次经过的结点（含根、叶结点）形成树的一条路径，最长路径的长度为树的深度。

问题分析

如果只有一个节点，则深度为1；
如果根节点只有左子树，则深度为左子树深度加1；
如果根节点只有右子树，则深度为右子树深度加1；
如果既有左子树又有右子树，则深度是左右子树中较大值加1；

Lowest Common Ancestor of a Binary Tree

问题描述

Given a binary tree, find the lowest common ancestor (LCA) of two given nodes in the tree.

According to the definition of LCA on Wikipedia: “The lowest common ancestor is defined between two nodes v and w as the lowest node in T that has both v and w as descendants (where we allow a node to be a descendant of itself).”

   _______3______
     /              \
 ___5__          ___1__
/      \        /      \
6      _2       0       8
      /  \
      7   4

For example, the lowest common ancestor (LCA) of nodes 5 and 1 is 3. Another example is LCA of nodes 5 and 4 is 5, since a node can be a descendant of itself according to the LCA definition.

问题分析

递归方式
边界条件：
如果root是null，空树，则返回root；
如果root和p或q任意一个相等，则root就是LCA；
如果给定root不是p或q节点中任何一个，则需要在root的左右子树中重新查找，此时有三种情况：
1、两个节点都在左子树上，右子树查找的公共节点是NULL；
2、两个节点都在右子树上，左子树查找出的公共节点是NULL；
3、一个在左子树，一个在右子树上，则当前节点root就是LCA；
