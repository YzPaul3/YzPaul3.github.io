Excel Sheet Column Title

问题描述

Given a positive integer,
return its corresponding column title as appear in an Excel sheet.

For example:

1 -> A
2 -> B
3 -> C
...
26 -> Z
27 -> AA
28 -> AB

问题分析

方法1、将A-Z存入一个数组，用数组下标与转化的数字进行对应
方法2、将数字看做字母的ASCII码值，其中A-Z为65-90，a-z为97-122
