常用初级排序

发表于 2016-04-04 | 分类于 数据结构

初级排序

冒泡排序

稳定，复杂度O(n2)

选择排序

不稳定，复杂度O(n2)
当前索引左边的所有元素都是有序的且位置确定。

插入排序

稳定，复杂度 O(n2)
当前索引左边的所有元素都是有序的，但是它们的最终位置不确定，当索引到达数组的右端是，排序就完成了。
插入排序对于部分有序的数组十分高效，也很适合小规模数组。
适用于顺序存储和链式存储的线性表。

快速排序

不稳定。

归并排序

稳定的排序算法。

链表中倒数第k个结点

问题描述

输入一个链表，输出该链表中倒数第k个结点。

问题分析

1、遍历链表两次，第一次统计出链表中结点的个数，第二次就能找到倒数第K个结点。
2、遍历链表一次，定义两个指针，让p1指针和p2指针都指向头结点，然后再让p2走(k-1)步，到达第k个节点。然后两个指针同时移动，当p2到达末尾的时候，p1所在位置就是倒数第k个节点了。

重建二叉树

问题描述

输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。

问题分析

在二叉树的前序遍历中，第一个数字总是根结点的值。但在中序遍历序列中，根结点的值在序列的中间，左子树的结点值位于根结点值的左边，右子树的结点值位于根结点值的右边。
先根据前序遍历序列的第一个数字创建根结点，接下来在中序遍历序列中找到根结点的位置，这样就能确定左右子树结点的数量。在前序遍历和中序遍历的序列中划分了左右子树的结点值之后可以递归的去构建它的左右子树。

合并两个排序的链表

问题描述

输入两个单调递增的链表，输出两个链表合成后的链表，当然我们需要合成后的链表满足单调不减规则。

问题分析

从两个链表的头结点开始合并，如果链表1的头结点的值小于链表2头结点的值，则链表1的头结点是合并后链表的头结点；如果链表2的头结点的值小于链表1头结点的值，则链表2的头结点是合并后链表的头结点。接下来继续合并两个链表中剩余的结点，合并的步骤和之前的一样。所以采用递归操作完成合并过程。

二叉树的下一个节点

发表于 2016-03-23 | 分类于 剑指Offer

二叉树的下一个节点

问题描述

给定一个二叉树和其中的一个结点，请找出中序遍历顺序的下一个结点并且返回。注意，树中的结点不仅包含左右子结点，同时包含指向父结点的指针。

问题分析

中序遍历二叉树的下一个结点，分为以下几种情况
1、二叉树为空，返回空；
2、若当前节点有右子树，则从右子树开始遍历，一直沿着左子结点进行查找，叶子结点就是下一个结点。
3、若当前节点没有右子树。则找到该结点的父节点，若该结点是父结点的左孩子，则下一个结点就是该结点的父结点。否则继续向上遍历父结点，直到找到结点是父结点的左孩子的结点，返回他的父结点。

数组中重复的数字

问题描述

在一个长度为n的数组里的所有数字都在0到n-1的范围内。 数组中某些数字是重复的，但不知道有几个数字是重复的。也不知道每个数字重复几次。请找出数组中任意一个重复的数字。 例如，如果输入长度为7的数组{2,3,1,0,2,5,3}，那么对应的输出是重复的数字2或者3。

问题分析

方法1：哈希表，时间O(n)，空间O(n)。
方法2：因为所有数字都在0到n-1的范围内，如果对数组进行排序，对于无重复的数组，数字和数组下标应该是一样的。利用这个想法，不用额外的空间，从头到尾依次遍历数组中的每个数字，将每个数字安排至指定的位置上去，如果不在就交换。如果在交换前发现，交换至的位置已经放置指定的数，则找到重复数字。详见算法2，时间O(n)，空间O(1)。

矩形覆盖

问题描述

我们可以用21的小矩形横着或者竖着去覆盖更大的矩形。请问用n个21的小矩形无重叠地覆盖一个2*n的大矩形，总共有多少种方法？

问题分析

可以横着放货竖着放，最后摆满的方式有两种，两个竖着放的矩形和2(n-2)的矩形拼接或者一个横着放的矩形和2(n-1)的矩形拼接。
即f(n)=f(n-1)+f(n-2)，转换为斐波那契数列。

发表于 2016-03-19 | 分类于 剑指Offer

变态跳台阶

问题描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

问题分析

方法1：递归
n级台阶，第一步有n种跳法：跳1级、跳2级、到跳n级
跳1级，剩下n-1级，则剩下跳法是f(n-1)
跳2级，剩下n-2级，则剩下跳法是f(n-2)
f(n)=f(n-1)+f(n-2)+…+f(1)
f(n-1)=f(n-2)+f(n-3)+…+f(1)
所以f(n)=2*f(n-1)

方法2：
移位操作
target=1时，有1种跳法1<<–1
target=2时，有2种跳法1<<–2
target=3时，有4种跳法1<<–3
target=4时，有8种跳法1<<–4
target=n时，跳法为1<<(n-1)

和为S的两个数字

问题描述

输入一个递增排序的数组和一个数字S，在数组中查找两个数，使得他们的和正好是S，如果有多对数字的和等于S，输出两个数的乘积最小的。
输出描述:
对应每个测试案例，输出两个数，小的先输出。

问题分析

因为已经排好序，首尾的数字向中间逼近，若和大于S，则尾标减1；若和小于S，则首标加1；若和等于S，则存入数组输出；且这两个数的乘积一定是最小的，即若x<y，x*y <= (x+a)(y-a);
